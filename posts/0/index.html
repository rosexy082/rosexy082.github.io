==会经行类型转换 ===不会经类型转换

eg: 12A123 parseInt()之后为12

其他类型转为boolean

​ Number 0 和NaN为false 其他均为true

​ String :空字符串为false,其他为true

​ Null和undefine:均为false

4.js的对象

1.Array对象用于定义数组

定义：

var 变量名 =new Array(元素列表);//方式一

var 变量名 =[元素列表];//方式二

访问：

arr[索引] =值;

特点：

类型可变

长度可变

遍历
属性：

length：设置或返回数组中元素的个数

方法：

forEach() 遍历数组中每一个有值的元素，每调用一次传入的参数

简化版

push()将元素添加到数组末尾，并返回新的长度

splice()从数组中删除元素

2.String 函数

定义

var 变量名 =new String(" … ");

方式二：

var 变量名 =" "

属性

方法

charAt() 返回指定位置的字符

indexOf() 检索字符串

trim() 去除字符串两边空格

substring() 提取字符串两个指定的索引号之间的字符(含头不含尾)
